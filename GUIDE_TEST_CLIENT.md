# 🎉 Guide de Test - BudgetWise
## Pour le Client Testeur

---

## 🎯 Bienvenue !

Merci de tester **BudgetWise**, votre application de gestion financière personnelle avec **Intelligence Artificielle** et **Gamification**.

---

## 🚀 Démarrage Rapide (5 minutes)

### 1️⃣ Créer Votre Compte
1. Ouvrir l'application
2. Cliquer **"S'inscrire"**
3. Remplir :
   - Nom : Votre nom
   - Email : votre@email.com
   - Mot de passe : (minimum 8 caractères)
4. Cliquer **"S'inscrire"**

✅ Vous êtes connecté automatiquement !

---

### 2️⃣ Créer Votre Premier Compte Bancaire
1. Aller sur **"Comptes"** (menu de gauche)
2. Cliquer **"+ Nouveau compte"**
3. Remplir :
   - Nom : **Compte Principal**
   - Type : **Compte Courant**
   - Devise : **XOF** (ou EUR, USD...)
   - Solde initial : **100000** (100,000 XOF par exemple)
4. Cliquer **"Créer"**

✅ Votre compte est créé !

---

### 3️⃣ Ajouter Vos Premières Transactions

#### Transaction 1 : Revenus (Salaire)
1. Aller sur **"Transactions"**
2. Cliquer **"+ Nouvelle transaction"**
3. Remplir :
   - Type : **Revenus** ✅
   - Compte : Compte Principal
   - Montant : **150000** (150,000 XOF)
   - Description : **Salaire Juin**
   - Catégorie : Salaire
   - Date : Aujourd'hui
4. Cliquer **"Créer"**

#### Transaction 2 : Dépense (Nourriture)
1. Cliquer **"+ Nouvelle transaction"**
2. Remplir :
   - Type : **Dépenses** ❌
   - Compte : Compte Principal
   - Montant : **35000** (35,000 XOF)
   - Description : **Courses supermarché**
   - Catégorie : Alimentation
   - Date : Aujourd'hui
3. Cliquer **"Créer"**

#### Transaction 3 : Dépense (Transport)
Répétez pour :
- Montant : **15000 XOF**
- Description : **Essence**
- Catégorie : Transport

✅ Vous avez maintenant un historique de transactions !

---

### 4️⃣ Créer un Budget

1. Aller sur **"Budgets"**
2. Cliquer **"+ Nouveau budget"**
3. Remplir :
   - Catégorie : **Alimentation**
   - Montant : **80000** (80,000 XOF/mois)
   - Période : **Mensuel**
4. Cliquer **"Créer"**

✅ Budget créé ! Vous verrez la progression en temps réel.

---

### 5️⃣ 🤖 Tester les Smart Alerts (IA)

1. Rester sur la page **"Budgets"** ou **"Smart Alerts"**
2. Cliquer sur **"🔔 Générer des Alertes"**
3. Attendre 2-3 secondes

✅ L'IA va détecter :
- Si vous dépassez vos budgets
- Des dépenses inhabituelles
- Des opportunités d'épargne
- La progression de vos objectifs

**Vous verrez des alertes colorées !**

---

### 6️⃣ Créer un Objectif d'Épargne

1. Aller sur **"Objectifs"**
2. Cliquer **"+ Nouvel objectif"**
3. Remplir :
   - Nom : **Vacances 2027**
   - Montant cible : **500000** (500,000 XOF)
   - Date limite : Dans 6 mois
   - Montant actuel : **0** (ou un montant déjà épargné)
4. Cliquer **"Créer"**

✅ Suivez votre progression avec une barre colorée !

---

### 7️⃣ 🎮 Débloquer des Achievements

Les achievements se débloquent automatiquement quand vous :
- ✅ Créez votre premier compte → **"Premier Pas"** 🥉
- ✅ Ajoutez 5 transactions → **"Organisé"** 🥈
- ✅ Créez un budget → **"Maître du Budget"** 🥇
- ✅ Respectez un budget → **"Discipliné"** 🥈
- ✅ Créez un objectif → **"Ambitieux"** 🥉

**Aller voir** : Menu **"Achievements"** pour voir vos récompenses !

---

### 8️⃣ 👥 Partager une Facture

1. Aller sur **"Partage de Factures"** (menu)
2. Cliquer **"+ Nouveau partage"**
3. Remplir :
   - Titre : **Resto entre amis**
   - Montant total : **30000** (30,000 XOF)
   - Description : Restaurant du vendredi
4. Ajouter participants :
   - Votre nom - 10,000 XOF
   - Ami 1 - 10,000 XOF
   - Ami 2 - 10,000 XOF
5. Cliquer **"Créer"**

✅ Vous pouvez maintenant marquer qui a payé et valider les paiements !

---

### 9️⃣ 📄 Exporter un Rapport PDF

1. Aller sur **"Transactions"**
2. Cliquer sur le bouton **"📥 Exporter PDF"**
3. Le PDF se télécharge automatiquement !

✅ Vous avez un rapport professionnel avec :
- Liste de toutes vos transactions
- Totaux (revenus, dépenses, solde)
- Logo BudgetWise

---

### 🔟 📱 Installer l'Application (PWA)

#### Sur Desktop (Chrome/Edge)
1. Regarder en haut à droite de la barre d'adresse
2. Cliquer sur l'icône **"⊕ Installer"** (ou attendre 30 secondes pour le popup)
3. Cliquer **"Installer"**
4. L'app s'ouvre dans sa propre fenêtre !

#### Sur Mobile (Android)
1. Menu Chrome → **"Ajouter à l'écran d'accueil"**
2. L'icône verte BudgetWise apparaît
3. Ouvrir → Expérience plein écran ! 📱

#### Sur iPhone
1. Safari → Bouton Partage
2. **"Ajouter à l'écran d'accueil"**
3. Fonctionne comme une app native !
